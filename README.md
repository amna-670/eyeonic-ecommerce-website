# Eyeonic E-Commerce Website

A responsive frontend e-commerce website built with **HTML, CSS, and JavaScript**, featuring a product carousel, mobile-friendly navigation, and clean UI components.

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive product carousel with thumbnail navigation
- Mobile navigation toggle (hamburger menu)
- Clean, modular HTML/CSS/JS structure

## 🛠️ Built With

- **HTML5** – page structure and content
- **CSS3** – styling, responsive layout, media queries
- **JavaScript (Vanilla)** – interactivity and DOM manipulation

## 📂 Project Structure

```
eyeonic-ecommerce-website/
│
├── index.html          # Main HTML file
├── style.css            # Stylesheet
├── script.js            # JavaScript functionality
└── assets/               # Images and other static assets
```

*(Update the structure above to match your actual folder/file names.)*

## 💻 Getting Started

### Prerequisites
Just a modern web browser — no build tools required.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/amna-670/eyeonic-ecommerce-website.git
   ```
2. Navigate into the project folder
   ```bash
   cd eyeonic-ecommerce-website
   ```
3. Open `index.html` in your browser

## 📱 Responsive Design

The website is optimized for all screen sizes using media queries, ensuring a smooth experience across mobile, tablet, and desktop devices.
